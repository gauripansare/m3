Move Files in Subfolders

In order to avoid any confusion over which images are to be used for which purpose, you want to keep the images for the brochure design (JPG images) and the images for the sponsor ads (PDF files) in separate folders for John.

Use the ribbon to create two new subfolders in the BFK_John folder, one named Brochure Design and one named Sponsor Ads. Move the files from John’s main folder into the appropriate subfolders. Note that the Word documents will stay in John’s main folder.
